# 三体文明Game

> 🚧 **项目正在开发中** 🚧

## 简介

《三体文明Game》是一款基于三体问题的太空模拟经营游戏。在这款游戏中，你将领导一个文明在充满不确定性的三体世界中生存与发展。

## 核心玩法

### 生存挑战
三体世界面临着永恒的混沌——三个恒星相互引力作用，行星轨道不可预测。你的文明必须适应极端多变的环境。

### 决策与发展
- **科技研发**: 开发新技术以应对各种危机
- **文明建设**: 建造设施，管理资源，照顾居民
- **能源利用**: 设计策略更大规模地利用三颗恒星的能量

### 进阶计划
- **流浪地球**: 改变行星轨道，寻找稳定的栖息地
- **末日预警**: 预测三体混沌，准备避难方案


## 运行方式

```bash
pip install -r requirements.txt
python main.py
```

## 技术栈

- Python 3
- Pygame
- NumPy
