# 三体文明游戏开发文档

## 项目概述

《三体文明》是一款基于刘慈欣《三体》世界观的三维星际经营模拟游戏。玩家需要在乱纪元中建立文明，发展科技，应对三体世界的不稳定环境。

### 技术栈
- **引擎**: Pygame 2.x + OpenGL (3D渲染)
- **语言**: Python 3.9+
- **配置**: YAML
- **数据**: JSON

### 行星区域系统

```
┌────────────────────────────────────────────┐
│          PlanetZoneManager                 │
│  72个区域 (12经度 × 6纬度)                  │
│                                            │
│  每帧更新:                                  │
│  1. 自转角度 += rotation_speed × dt         │
│  2. 对每个区域:                              │
│     a. 计算法线 normal(lat, lon+rotation)   │
│     b. 对每颗恒星:                           │
│        - direction = normalize(star - planet)│
│        - cos_angle = dot(normal, direction)  │
│        - 光照 = cos_angle × mass / dist²    │
│        - 辐射 = cos_angle × mass / dist^2.5 │
│     c. 温度 = -273.15 + Σ(光照贡献 × 50)    │
│  3. 全球平均 → 主界面显示                     │
│  4. 逐区数据 → 区域浏览/建筑伤害             │
└────────────────────────────────────────────┘
```

```
Tier0              Tier1              Tier2              Tier3         Tier4

基础冶金 ────┬── 实验室 ────────┬ 应用物理 ──── 科学院 ──── 可控核聚变
             └── 发电站         └ 材料科学 ──── 高强度合金

```

科技点数类型：
- **基础科研** — 人口每500人自动每天产出1点
- **应用科研** — 需建造「实验室」，每座每天产出2点，消耗8能源/天
- **理论科研** — 需建造「科学院」，每座每天产出1点，消耗25能源+5食物/天

## 模块使用指南

### ScreenManager 使用

```python
from ui.screen_manager import ScreenManager, ScreenType

# 获取单例实例
manager = ScreenManager()

# 注册界面
manager.register_screen(ScreenType.SETTINGS, settings_screen)

# 切换界面（自动压栈）
manager.switch_to(ScreenType.SETTINGS)

# 返回上级界面
manager.go_back()  # 或指定 fallback: go_back(ScreenType.INITIAL_MENU)

# 清空栈并切换（用于退出到主菜单）
manager.clear_stack_and_switch(ScreenType.INITIAL_MENU)
```

### 设置系统使用

