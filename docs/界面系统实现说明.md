# 游戏界面系统实现说明

## 已实现功能

### 1. 界面管理核心 (ui/screen_manager.py)

**ScreenManager** - 单例模式的界面管理器
- 管理所有界面的注册和切换
- 维护界面栈，支持返回功能
- 全局状态管理（游戏是否开始、当前存档槽位等）

**Screen** - 界面基类
- 提供生命周期方法：on_enter, on_exit
- 支持进入/退出动画
- 统一的事件处理和渲染接口

**ScreenType** - 界面类型枚举
- INITIAL_MENU: 初始菜单
- START_GAME_MENU: 开始游戏菜单
- SETTINGS: 设置界面
- GAME_MENU: 游戏内菜单
- MAIN_SCREEN: 2D主游戏界面（待整合）
- STARMAP_VIEW: 3D星图界面（待整合）

### 2. 初始菜单 (ui/initial_menu.py)

**InitialMenu** - 游戏启动后的第一个界面
- 动态星空背景（带视差和闪烁效果）
- 游戏标题 "三体文明"（带发光效果）
- 三个主按钮：开始游戏、设置、退出
- 按钮悬停动画（向右偏移）
- 版本号显示

### 3. 开始游戏菜单 (ui/start_game_menu.py)

**StartGameMenu** - 新游戏/继续游戏/加载存档
- 四个选项按钮：新游戏、继续游戏、加载存档、返回
- 存档槽位显示（6个槽位，2x3布局）
- 显示存档信息：保存时间、游戏天数
- 点击存档槽位加载对应游戏

**SaveSlot** - 存档槽位组件
- 自动检测存档文件
- 显示存档元数据
- 选中和悬停状态

### 4. 设置界面 (ui/settings_screen.py)

**SettingsScreen** - 游戏各种设置
- 四个标签页：游戏、显示、音频、控制
- 滑块组件用于数值设置
- 当前实现的设置项：
  - 游戏：时间流逝速度、自动保存间隔
  - 音频：主音量、音乐音量、音效音量
  - 控制：鼠标灵敏度
- 设置保存到 data/settings.json
- 应用和重置按钮

**GameSettings** - 设置数据类
- 使用dataclass定义所有设置项
- 支持序列化到JSON
- 提供默认值

**SettingSlider** - 设置滑块组件
- 可拖动的滑块手柄
- 实时更新值
- 可配置的数值范围和精度

### 5. 游戏内菜单 (ui/game_menu.py)

**GameMenu** - 游戏过程中按ESC打开的菜单
- 半透明黑色背景遮罩
- 五个选项：继续游戏、设置、保存游戏、返回主菜单
- 快速保存功能（保存到当前槽位）
- 进入/退出动画效果

## 整合到主循环

### 初始化
```python
# 创建界面管理器
screen_manager = init_screen_manager(screen)

# 切换到初始菜单
screen_manager.switch_to(ScreenType.INITIAL_MENU)
```

### 主循环结构
```python
while running:
    dt = clock.tick(fps) / 1000.0
    
    # 更新当前界面
    screen_manager.update(dt)
    
    # 处理事件
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
        else:
            screen_manager.handle_event(event)
    
    # 渲染
    screen.fill((10, 10, 20))
    screen_manager.render(screen)
    pygame.display.flip()
```
