# 三体文明游戏设计文档

## 界面与交互设计

### 1. 2D主界面新增功能
在游戏正式的2D主界面中，计划新增以下交互按钮：
* **科技树按钮**：点击后将进入一个全新的科技树专属界面（类似于现在的星图切换方式）。
* **政策系统按钮**：点击后可进行文明级的宏观政策调控，包含“脱水”、“浸泡”、“开启大生育计划”等特殊行动，以应对多变的乱纪元与恒纪元环境。

## 游戏系统设计

### 1. 科技树系统
* **专属子界面**：包含一棵完整的展现文明发展路径的科技树视图。
* **研发条件限制**：解锁和研发新科技需要满足一定的前提条件，例如消耗特定数量的“人口”、达到某种“环境”指标或积累了足够的进度。

### 2. 星图与观测系统（渐进式探索）
* **初期不可见**：游戏开始时，受限于文明的观测能力，星图系统不能被直接打开。
* **通过科技解锁**：
  * **观测星图**：玩家必须在科技树中研发出“望远镜”相关科技，并且拥有足够的“科学”属性，才能够解锁并观察星空。
  * **轨道预测**：随着后续科技进步，当研发出“计算机技术”后，星图功能得到加强，能够对三体系统的天体运动实现一定的“轨道预测”功能。
