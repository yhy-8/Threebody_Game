# 三体文明游戏设计文档

## 一、界面与交互体验设计

* **数据告警效果**：当"表面温度"过热过低，或"社会安定度"低于一定阈值时，相关字体颜色不仅会变红，还要加入轻微的闪烁（Flicker）效果与全局的红色警报滤镜。
* **信息高亮提示（Tooltip）**：鼠标悬浮在某个资源或参数上时，浮现出半透明说明框，解释该数据的变化来源。
* 未来会进一步在主界面加深粒子星空背景的随机变化，以应对环境状态改变时的直观情绪传达。
* **区域浏览界面**：2D Mercator投影风格的行星展开图，12×6网格热力图显示温度/辐射/光照，点击区域查看详细数据和建筑列表。

## 二、行星区域系统

### 1. 球体分区
行星被划分为 **72个区域**（12经度带 × 6纬度带），每个区域覆盖30°经度 × 30°纬度的球面。

- 极地区域面积小（cos权重），赤道区域面积最大
- 每个区域有随机地形类型（平原、高原、山地、峡谷、盆地、丘陵）
- 地形决定区域的资源禀赋：铁矿、铜矿、稀有矿、肥沃度、藻类密度
- 区域温度影响区域内工人的工作效率（过高过低都会导致效率下降甚至停工）
- 区域面积权重影响全球平均值的计算

### 2. 行星自转
- 自转速度：15°/游戏天
- 自转角度影响每个区域的法线方向
- 面朝恒星的区域接受直射辐射，背面区域仅有5%散射辐射
- 温度具有热惯性，不会瞬变

### 3. 逐区环境计算
每帧对每个区域：
1. 计算区域法线 `normal = sphere_normal(lat, lon + rotation_angle)`
2. 对每颗恒星计算方向和距离
3. 光照 = `max(0, dot(normal, star_dir)) × mass × 10 / (dist² + 100)`
4. 辐射 = 类似计算，衰减更快（dist^2.5）
5. 温度 = `-273.15 + Σ(光照贡献 × 50)`

### 4. 建筑与区域
- 建筑有所在区域ID
- 建筑有耐久度、热/冷/辐射抗性
- 区域温度/辐射超过抗性阈值时，建筑受损
- 耐久度归零时建筑损毁
- 建筑的产出由分配的工人数量、工种的基础产出、电力负载平衡、科技自动化倍率以及温度效率综合决定
- 科技可强化建筑抗性（防辐射装甲、高强度合金等）

## 三、科技树系统（缺氧式研究分级）

### 1. 三种科技点数
| 类型 | 来源 | 产出条件 |
|------|------|----------|
| 基础科研 | 人口自然产出 | 每500人口/天产出1点 |
| 应用科研 | 实验室建筑 | 每座实验室/天产出2点，消耗8能源/天 |
| 理论科研 | 科学院建筑 | 每座科学院/天产出1点，消耗25能源+5食物/天 |

> 类比《缺氧》：简单科技只需基础研究台（对应基础科研），复杂科技需要超级计算机（对应科学院产出理论科研）。
> 高级研究建筑对电力和食物的消耗极大。

### 2. 科技树结构（5层17节点）
- **Tier 0**：望远镜、维生庇护所、基础冶金、基础农业
- **Tier 1**：计算机技术、天文观测站、深地庇护所、实验室、发电站
- **Tier 2**：混沌预测模型、自动化控制、防辐射装甲、应用物理、材料科学
- **Tier 3**：科学院、高强度合金
- **Tier 4**：可控核聚变

### 3. UI实现
- 树形可视化：按层级从左到右排列，贝塞尔曲线连接依赖关系
- 三种节点状态：锁定(灰)、可研发(蓝)、已解锁(绿)
- 鼠标悬浮Tooltip：详细显示所有需求和效果
- 支持缩放(滚轮)和拖拽(右键)浏览大型科技树
- 顶部实时显示三种科技点数余额

## 四、决策系统（原"政策系统"重构）

决策系统分为两大类：

### 1. 建筑建造决策
选择要建造什么建筑，指定建造区域，消耗相应资源：

| 建筑 | 资源消耗 | 科技需求 | 工人容量 | 效果 |
|------|---------|---------|------|------|
| 藻类采集场 | 20铁 | 无 | 5 | 藻类燃料产出 |
| 铁矿场 | 30铁 | 无 | 5 | 铁矿产出 |
| 铜矿场 | 40铁+10铜 | 基础冶金 | 5 | 铜矿产出 |
| 稀有矿场 | 60铁+20铜 | 基础冶金 | 3 | 稀有矿物产出 |
| 农场 | 50铁 | 基础农业 | 8 | 食物产出 |
| 化石燃料井 | 80铁+20铜 | 化石燃料开采 | 5 | 化石燃料产出 |
| 藻电站 | 60铁+15铜 | 基础电气化 | 3 | 消耗藻燃料发15kW电 |
| 火力发电站 | 200铁+100铜 | 发电站科技 | 3 | 消耗化石发45kW电 |
| 庇护所 | 100铁 | 维生庇护所 | 0 | 耗电1kW,区域保护+20% |
| 实验室 | 200铁+80铜 | 实验室科技 | 5 | 耗电8kW,产出应用科研 |

| 辐射屏蔽站 | 300铁+20稀有 | 防辐射装甲 | 0 | 耗电5kW,辐射抗性+50% |

### 3. 人力分配与资源机制
- **人口驱动模型**：所有的生产活动都需要分配人口到对应岗位（铁矿工、铜矿工、电厂操作员、科学家等）。
- **细分资源**：
  - 矿物类：铁（基础材料）、铜（电气材料）、稀有矿物（高级科技）
  - 能源类：藻类燃料（初级）、化石燃料（中级）
  - 二次能源：电力（kW，建筑维持需要耗电，电力不足全境停电减产）
  - 维生：食物（不足将导致人口饿死）
- 玩家需要在“人口管理”界面动态分配空闲人口。建筑提供的“岗位上限”决定了最多能分配多少工人，而实际产出受岗位填满度影响。

### 2. 文明政策决策
- **全民脱水**：应对极端环境，资源消耗-80%，产出-90%
- **浸泡复苏**：恢复正常状态
- **大生育计划**：消耗500食物，人口增长+200%
- **配给制**：食物消耗-50%，效率和稳定性下降
- **工业强心剂**：产出+150%，健康和稳定性大幅下降

## 五、星图与渐进式探索
* **初期盲区**：游戏开始时，星图系统被锁定。
* **视野解锁**：研发"望远镜"科技解锁星图。
* **逐步预测**：研发"计算机技术"、"混沌预测模型"后，可预测行星未来轨道。
* **行星区域可视化**：星图中行星显示经纬网格线，随自转实时旋转。
* **行星锁定视角**：可锁定行星进行环绕观察，近距离观察区域分布。
